# Vue boilerplate

Vue+Vuetify 사용

* 테스트 서버 실행
   ```
   yarn serve
   ```

* 빌드
   ```
   yarn build
   ```

* 개발빌드
   ```
   yarn dev-build
   ```

* 린트
   ```
   yarn lint
   ```

* 테스트 - 모카 사용
   ```
   yarn test:unit
   ```
